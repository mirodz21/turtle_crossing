# 🐢 Turtle Crossing

**Turtle Crossing** is a simple arcade-style game built with Python.  
Your goal is to help the turtle cross a busy street without getting hit by passing cars. The further you go, the harder it gets!

---

## 🎮 Features

- Classic arcade-style gameplay  
- Increasing difficulty as levels progress  
- Randomly generated moving cars  
- Score/level tracking  
- Collision detection system  
- Simple and beginner-friendly Python structure
